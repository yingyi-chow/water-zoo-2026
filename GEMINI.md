# Gemini CLI Capabilities for App Creation

As your Gemini CLI agent, I can assist you in building, refactoring, and maintaining this application using the following key capabilities:

## 1. Codebase Exploration & Analysis
*   **Investigate Codebase:** I can analyze the entire architecture and dependencies to understand complex systems before making changes.
*   **Search Content:** I can perform fast, optimized searches (`ripgrep`) across the project to find specific variables, functions, or patterns.
*   **Find Files:** I can quickly locate files using glob patterns (e.g., `**/*.tsx`).
*   **Read & List:** I can read file contents and list directory structures to understand the local context.

## 2. Code Editing & Generation
*   **Write Files:** I can create new files or overwrite existing ones with generated code, configuration, or documentation.
*   **Targeted Replacements:** I can perform precise, surgical edits within existing files by replacing specific strings of code, ensuring I only modify what's necessary.

## 3. Environment & Execution
*   **Run Shell Commands:** I can execute bash commands on your system. This allows me to:
    *   Install dependencies (e.g., `npm install`).
    *   Run build scripts (e.g., `npm run build`, `npm run dev`).
    *   Execute tests and linting (e.g., `npm test`, `npm run lint`).
    *   Start development servers.
    *   Manage Git version control (e.g., `git status`, `git commit`).

## 4. External Information & Web Access
*   **Google Web Search:** I can search the web for API documentation, solutions to errors, or library best practices.
*   **Fetch Web Content:** I can read content directly from URLs (including `localhost` or remote URLs) to parse documentation or external data relevant to your app.

## 5. Planning & Memory
*   **Task Management (Todos):** For complex requests, I can create and manage a dynamic subtask list to keep track of progress and ensure no steps are missed.
*   **Long-term Memory:** I can save specific facts about your preferences, environment, or project constraints to personalize future interactions.

## How to use me effectively:
*   Be specific about *what* you want to achieve (e.g., "Add a user authentication flow using Firebase").
*   Ask me to add features, fix bugs, explain code, refactor components, or write unit tests.
*   If a task is complex, I will propose a plan or create a todo list for you to review as we progress.
